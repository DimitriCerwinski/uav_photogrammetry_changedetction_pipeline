# Untersuchung, wie aus Drohnenbilddaten digital 3D-Modelle erzeugt und Veränderungen zwischen verschiedenen Erfassungszeitpunkten effizient erkannt und visualisert werden können 

Dieses Repository enthält den im Rahmen meiner Bachelorarbeit entwickelten Pipeline-Code.

## Externe Abhängigkeiten

Der Code verwendet externe Bibliotheken, darunter insbesondere:

- pycolmap / COLMAP 
- TEASER++		

Externe Bibliotheken sind nicht Bestandteil dieses Repositories.  
Die verwendeten Versionen sind in der Arbeit dokumentiert.

## Hinweis zur Erstellung

Bei der Entwicklung dieses Codes wurde KI-gestützte Unterstützung verwendet. Anpassung, Validierung und finale Umsetzung des enthaltenen Codes erfolgten durch den Autor.
